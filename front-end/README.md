
## Requirements
- Java
- NodeJs
- Elm
- Yarn (optional)

## Building

```
$ elm-make
$ npm install
```

## Running
For production
```
$ npm run build
```

For development
```
$ npm run start
```
loads on http://0.0.0.0:8080


## Preparing 

1. Install NodeJs
```
$ dnf install nodejs
```
you'll have `npm` tool, packager for NodeJs platform.

2. Install Elm

```
$ npm install -g elm
```
After installing available tools elm-repl, elm-reactor, elm-make, elm-package
